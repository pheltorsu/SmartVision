SmartVision Detection System

Welcome to SmartVision an intelligent detection system designed to enhance computer vision capabilities for real-world applications. This system integrates Object Detection, Lane Detection, and Curve Detection into one platform.

It can analyze live video streams from a camera or process uploaded videos to:

  • Identify and label objects using YOLOv8 AI model.
  • Detect road lanes and curves using OpenCV image processing.
  • Provide a clear, real-time visual analysis of detected features.
